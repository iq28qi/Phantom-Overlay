# assets/

Папка для визуальных ассетов, используемых в README и на GitHub-странице релизов.

## Что сюда класть

| Файл | Описание | Когда использовать |
| --- | --- | --- |
| `preview.png` | Основной скриншот оверлея (prestige shot). | Hero-изображение в README. |
| `overlay_in_game.gif` | Короткий GIF — оверлей поверх игры (CS2 / рабочего стола). | Раздел «как выглядит в бою». |
| `settings_live.gif` | GIF работы Live Preview в Control Center при изменении настроек. | Раздел «Control Center». |
| `presets.png` | Коллаж/скриншот панели пресетов (Neon Mint / Ultraviolet / …). | Раздел «Presets». |
| `icon.svg` | Логотип проекта в векторе (если появится). | Заголовок README. |

## Рекомендации по формату

- **GIF** — до 10 МБ каждый, не длиннее 10 секунд, 24–30 fps. Для записи подойдёт `ScreenToGif` или `LICEcap` на Windows.
- **PNG** — скриншоты в исходном разрешении (не масштабировать вниз), с прозрачностью там, где это уместно.
- Имена файлов — `snake_case`, без пробелов и кириллицы.

Ссылки в `README.md` указывают на `assets/<имя_файла>` через относительный путь, так что после добавления картинки она подтянется автоматически.
