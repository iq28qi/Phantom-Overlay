# 👻 Phantom Overlay 4.0 — Ultra Edition

**Phantom Overlay** — это современный, легковесный и полностью настраиваемый внутриигровой оверлей для Windows. Разработан на Python с использованием библиотеки PyQt6 и выполнен в премиальном стиле **Glassmorphism** (эффект матового стекла).

Программа выводит критически важную информацию о системе поверх любых окон, обладает встроенным AI-ассистентом и умеет автоматически скрываться, когда вы находитесь вне игры.

## ✨ Главные возможности

### 📊 Продвинутый мониторинг
* **Hardware Info:** Отслеживание температуры и загрузки GPU (NVIDIA), а также использования CPU и RAM в реальном времени.
* **Network Status:** Постоянный мониторинг сетевой задержки (Ping) для контроля лагов в соревновательных играх.

### 🤖 AI-Ассистент (Silphiette)
* **Голосовые предупреждения:** Ассистент непрерывно следит за состоянием железа. Если температура видеокарты превышает критическую отметку (82°C), Silphiette мгновенно сообщит об этом голосом.
* **Текстовый статус:** Отображение текущего состояния системы и медиа в интерфейсе оверлея.

### 🎮 Игровые функции и Интеграция
* **Smart Focus (Авто-скрытие):** Оверлей автоматически появляется только при запуске целевых игр (например, `CS2`, `Dota 2`, `Genshin Impact`, `Minecraft`, `GTA 5 RP`, `Cyberpunk 2077`) и исчезает на рабочем столе.
* **Discord Rich Presence:** Трансляция статуса вашего железа (температуры) и текущего музыкального трека прямо в ваш профиль Discord.
* **Управление медиа:** Перехват данных о текущем треке через Windows SDK. Управление музыкой (Play/Pause, Next Track) доступно прямо из системного трея.

### 🎨 Кастомизация (Control Center)
* **Вкладочный интерфейс:** Удобное меню настроек, разделенное на логические категории.
* **Glassmorphism Design:** Настройка уровня аппаратной прозрачности окна.
* **Умные обои:** Поддержка пользовательских фоновых изображений (`.png`, `.jpg`), которые автоматически адаптируются под скругленные углы оверлея.
* **Динамическая иконка:** Просто положите любую картинку `icon.png` в папку с программой, и она автоматически станет иконкой приложения.

## 🚀 Инструкция по запуску (Для пользователей)

Если вы просто хотите использовать оверлей во время игры, вам **не нужно** устанавливать Python или лезть в код.

1. Перейдите в раздел **[Releases](../../releases)** в правом меню GitHub.
2. Скачайте последнюю версию файла `PhantomOverlay.exe`.
3. Поместите файл в любую удобную папку (например, `C:\Phantom`).
4. **(Опционально):** Положите в эту же папку картинку с названием `icon.png`, чтобы изменить иконку в системном трее.
5. Запустите `PhantomOverlay.exe`.

Оверлей плавно появится на экране, а в системном трее (возле часов) появится значок призрака.

## ⚙️ Управление и Настройки

Управление программой осуществляется через **Системный трей Windows** (значок в правом нижнем углу) или с помощью горячих клавиш.

* **Правый клик по значку в трее:** * `🛠 Режим перетаскивания` — разблокирует окно, позволяя переместить его в любую точку экрана.
  * `⏯ Управление музыкой` — переключение треков прямо из меню.
  * `⚙ Настройки` — открывает Phantom Control Center.
* **Глобальный Хоткей:** По умолчанию `Ctrl + Shift + P` — моментально скрывает или показывает оверлей поверх всех окон (сопровождается плавной анимацией). Хоткей можно изменить в настройках.

*Примечание: Все ваши настройки автоматически сохраняются в файл `phantom_config.json` в папке с программой.*
